open fun lltDecompose(lowerTriangular: Boolean): Matrix<R, C>?
Content copied to clipboard

Decompose "this" matrix using Cholesky Decomposition. If the "this" matrix is zeros, it will return the zero matrix.

Return

The decomposed matrix.

Parameters

lowerTriangular

Whether or not we want to decompose to the lower triangular Cholesky matrix.

Throws

kotlin.RuntimeException

if the matrix could not be decomposed(ie. is not positive semidefinite).
