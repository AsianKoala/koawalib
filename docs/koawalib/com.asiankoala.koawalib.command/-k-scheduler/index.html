object KScheduler
Content copied to clipboard

CommandScheduler runs all commands. Every loop the scheduler checks for newly scheduled commands, cancelled commands, finished commands, and handles them accordingly. Processing is done behind the scenes, so the main purpose of this class for the user is to schedule commands, mainly using KScheduler.schedule
